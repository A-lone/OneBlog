
## OneBlog
构建于Asp.net MVC框架的博客平台

## Environment
  * Visual Studio 2015 +
  * .Net Framework 4.5.2
  * ASP.NET MVC 5 +

## Get started
  1. Clone repository
  2. Open solution in Visual Studio 2015
  3. Builld and run solution to load website in the browser
  4. Use admin/admin to log into website administration
